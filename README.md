# bad_gopher

Requires: ffmpeg, yt-dlp (optional)

Usage: `bad_gopher -p <path> -u <url> -m <true/false> -a <true/false> -f <fps>`

Default options:

- p: resources/input.mp4

- u: NONE

- m: true

- a: true

- f: -1 \(Automatic)

Put an mp4 file in ./resources named input.mp4 and run the program.
